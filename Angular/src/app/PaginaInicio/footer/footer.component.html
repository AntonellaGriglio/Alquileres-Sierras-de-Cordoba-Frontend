Preguntas Frecuentes

Para realizar una consulta de disponibilidad, se requiere que se ingresar como minimo nombre, apellido, telefono , email, la fecha de ingreso y egreso. Esos son los datos obligatorios para que la consulta se pueda realizar correctamente.

Pedimos esa información para saber desde donde nos visitan nuestros clientes.

Para realizar una consulta se tiene que llenar el formulario y al enviarlo correctamente, se redirecciona al WhatsApp del complejo que le interesa.
